Important Gaming terms

  • Chat/Voice chat terms
  • RNG - Stands for Random Number Generator. Usually a synonym for luck in games
  • Throwing - usually means that the person is losing the game in some way
  • Trolling - Intentionally ruining some part of the game for humor
  • Micro - stands for micromanagement, usually some sort of metric for how efficient you are.
  • Co-Op - Stands for cooperative, working together for some sort of common goal in a game.
  • Game types
  • MMO - Stands for Massive Multiplayer Online. Signifies a big game usually with a connected world
  • RPG - Stands for Role-Playing game. Sometimes combined with MMO's to make an MMORPG(essentially a connected world RPG game
  • Action - Usually just games with some sort of combat system that the player has to deal with.
  • FPS - Stands for First-Person Shooter. Usually based on some sort of combat with weapons, and usually highly competitive.
  • Roguelike - Essentially a dungeon run game. Usually there is very little permanent progress in these types of games.
  • Sandbox - Essentially a game where a lot of the gameplay is made by the player. For example, a game like Minecraft is a sandbox game as you can do whatever you want.
  • Open-World games - Means that the entire world is open for you to explore, usually less streamlining and more exploration.
  • Tower defense - Usually smaller games that rely on you placing defenses against waves of enemies.
  • Turn-Based - Games that are turn based aren't based off of reactions - everyone has a turn to go, usually in an order.
  • RTS - Stands for Real-Time Strategy. Usually heavily based on micromanegment.
  • General computer terms used in gaming
  • PC-Stands for Personal Computer, games on your computer.
  • Console - Any sort of console such as an Xbox, PS4, etc. Generally means a console connected to a TV.
  • Mobile - Any game based on some sort of mobile device, like your phone.
